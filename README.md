# Servidor Socket TCP em Java

Este projeto demonstra a implementação de um **servidor TCP** em Java, que aguarda conexões de clientes através de sockets.  
O servidor recebe mensagens enviadas por um cliente conectado e as exibe no console até que a mensagem `"fim"` seja recebida, encerrando a comunicação.

## 🚀 Tecnologias utilizadas
- Java (Socket, PrintWriter, IO)
